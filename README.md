# WordCount
A Big Data related project. "\n" Part 1: Write JAVA code to download and decompress files, upload them onto HDFS, and write WordCount Algorithm to count each word in all the six books. "\n" Part 2: Use Twitter Search API to gather tweets about a topic on 6 different timelines and write a WordCount Algorithm to count each occurrence of a hashtag. "\n" Technologies: Java, MapReduce
